
For creating a new volume of tambora.org data serie, please update:



variables.tex -> depends on authors, title etc.

data.tex -> will be generated from a collection by tambora.org 

materialspecification.tex -> needs to be written from authors of collection

fig/... .pdf -> needs to be calculated by r-scripts. For more informations, please check https://github.com/tambora-org/tamboRapi/tree/master/R
